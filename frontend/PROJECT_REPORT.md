# SYSU Image Hunt 前端开发报告

## 1. 项目概览
**SYSU Image Hunt** 是一款基于校园场景的寻宝/捉迷藏类 Web 应用。本项目目前完成了前端核心页面的开发，实现了高保真的 UI 交互和基础的业务逻辑模拟。

## 2. 技术栈与设计理念

### 2.1 技术栈
- **核心框架**: React 19 + Vite
- **路由管理**: React Router DOM v7
- **样式方案**: Tailwind CSS v4 (原子化 CSS)
- **动画库**: Framer Motion (页面转场与微交互)
- **图标库**: Lucide React

### 2.2 设计理念
- **Apple-style Aesthetic**: 追求极致的简洁与高级感，采用大圆角、平滑的过渡和细腻的阴影。
- **Glassmorphism (玻璃拟态)**: 大量使用背景模糊 (`backdrop-blur`) 和半透明背景，营造出层次感和现代感。
- **深色模式优先**: 默认采用沉浸式的深色背景，同时支持一键切换至浅色模式，适配不同用户偏好。

## 3. 核心功能与实现细节

### 3.1 用户认证 (Login/Register)
- **功能**: 支持账号登录、注册及密码重置流程。
- **实现**:
    - 使用 `framer-motion` 实现 Tab 切换时的平滑过渡。
    - 动态改变提交按钮文本（"登录" vs "注册"）。
    - 模拟的密码重置流程，包含倒计时和自动跳转。

### 3.2 仪表盘 (Dashboard)
- **功能**: 用户的个人中心，展示用户信息、战绩统计及核心功能入口。
- **实现**:
    - 采用 Grid 布局展示统计卡片。
    - 底部导航栏提供快速入口。
    - 实现了与“个人设置”和“排行榜”页面的无缝导航。

### 3.3 游戏菜单 (Game Menu)
- **功能**: 选择游戏模式（普通匹配 vs 排位赛）。
- **实现**:
    - **卡片式设计**: 使用大尺寸卡片展示不同模式，Hover 时触发缩放和光效动画。
    - **参数化路由**: 点击不同模式时，通过 URL Query 参数 (`?mode=normal` 或 `?mode=ranked`) 传递模式信息给匹配页面。

### 3.4 匹配系统 (Matchmaking)
- **功能**: 模拟匹配过程，展示对手信息和倒计时。
- **实现**:
    - **动态标题**: 根据 URL 参数判断是“普通匹配”还是“排位赛”，并展示对应标题。
    - **倒计时逻辑**: 使用 `setInterval` 模拟 3 秒倒计时，结束后自动跳转至游戏页。
    - **视觉效果**: 动态的 VS 动画和渐变背景。

### 3.5 个人设置 (Settings)
- **功能**: 修改个人信息、切换主题、退出登录。
- **实现**:
    - **主题切换**: 利用 `localStorage` 持久化存储用户的主题偏好 (`darkMode`)。通过 `useEffect` 监听状态变化，动态向 `body` 添加或移除 `light-mode` 类名，从而触发 Tailwind 的样式切换。
    - **数据模拟**: 昵称修改目前仅在前端 State 中生效，演示了表单交互。

### 3.6 排行榜 (Leaderboard)
- **功能**: 展示好友和全服排名。
- **实现**:
    - 实现了 Tab 切换查看不同维度的榜单。
    - 高亮显示当前用户在榜单中的位置。

## 4. 演示与验证流程

您可以按照以下路径体验完整流程：

1.  **登录**: 进入首页 -> 输入任意账号密码 -> 点击登录。
2.  **查看设置**: 点击右上角“个人设置” -> 尝试切换“深色模式” -> 点击“保存”。
3.  **查看排行**: 在仪表盘点击“查看排行榜” -> 切换“好友/全服”标签。
4.  **开始游戏**:
    - 点击“开始游戏”进入菜单。
    - 选择“积分排行” -> 确认页面显示“排位赛匹配成功”。
    - 等待倒计时结束进入游戏页面。

## 5. 后续规划 (Backend Integration)

当前版本为纯前端实现，后续需对接后端 API 以实现完整功能：
- **用户数据**: 对接 `/api/user/profile` 获取真实的昵称、头像和积分。
- **排行榜**: 对接 `/api/leaderboard` 获取实时排名数据。
- **匹配系统**: 对接 WebSocket 服务，实现真实的玩家实时匹配。
- **游戏逻辑**: 对接地图数据和实时位置同步。
